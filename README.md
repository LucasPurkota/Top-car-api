# TopCar API

TopCar API is a car e-commerce application developed in next.js.

## Get Starting

1. Clone the repository.
2. install modules.
```bash
npm install
```
3. Run aplication.
```bash
npm run dev
```
Open http://localhost:3000 with your browse to see result.
